# Project Overview

## What is Market Analyst AI?

Market Analyst AI is an autonomous, multi-agent research system designed to generate comprehensive market reports. By leveraging Google's Gemini models and real-time web data via Brave Search, it orchestrates a team of specialized AI agents to plan, research, analyze, and synthesize actionable market intelligence.

## Core Purpose

To democratize access to high-quality market research by automating the traditional analyst workflow, providing users with real-time, data-driven insights in a fraction of the time required for manual research.

## Technology Stack

- **Backend**: Python 3.9+, FastAPI, Server-Sent Events (SSE)
- **Frontend**: Node.js 18+, Next.js (React), TypeScript, Tailwind CSS
- **AI/ML**: Google Gemini API (`gemini-2.0-flash`)
- **Search**: Brave Search API / Tavily API
- **Infrastructure**: Docker, Docker Compose

## Key Features

- **Multi-Agent Pipeline**: distinct roles for Strategy, Research, Analysis, and Synthesis.
- **Real-Time Streaming**: Live updates via SSE protocol.
- **Deep Web Research**: Integration with Brave and Tavily for live data access.
- **Structured Reporting**: Generates reports with metadata, key metrics, and markdown formatting.
- **Observability**: Prometheus metrics endpoint for monitoring agent performance.

## Getting Started

1. **Configure Environment**:
   - Copy `.env.example` to `.env` in `backend/` and set `GOOGLE_API_KEY`.
   - Copy `.env.local.example` to `.env.local` in `frontend/`.

2. **Run Backend**:
   ```bash
   cd backend && python simple_agent.py
   ```

3. **Run Frontend**:
   ```bash
   cd frontend && npm run dev
   ```

4. **Access**: Open [http://localhost:3000](http://localhost:3000)
